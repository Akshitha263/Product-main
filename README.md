**ProductApp**
**Project Overview**
ProductApp is an Android application developed using Kotlin and Jetpack Compose.
The application demonstrates modern Android development practices such as MVVM architecture, navigation between screens, and ViewModel state management.
The app allows users to navigate through product-related screens while maintaining application state efficiently.

**Features**
Modern UI built with Jetpack Compose
Navigation between screens using Navigation Component
State management using ViewModel
Clean architecture structure
Simple and user-friendly interface

**Technologies Used**

Kotlin
Android Studio
Jetpack Compose
MVVM Architecture
Android Navigation
ViewModel
